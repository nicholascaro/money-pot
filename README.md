### How To Run

#### Requirements
- npm
- maven
  - you can install via homebrew `brew install maven`

##### How to Run Client
1. Install dependecies fron package.json `npm init`
2. `cd client`
3. `npm start`

##### How to Run Server
1. `cd server`
2. `mvn spring-boot:run`
